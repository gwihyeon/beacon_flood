# Beacon Flooding Attack
Tested in Kali Linux 2022.4  

## Installation
```sh
pip3 install scapy
pip3 install randmac
```

## Usage
python3 beacon_flood.py [interface_name] [ssid_list_file_name]  
ex) python3 beacon_flood.py wlan0 ssid_list.txt
